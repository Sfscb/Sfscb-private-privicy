S.F.S.C.B Privacy Policy

Effective Date: February 10, 2025

1. Data Collection

S.F.S.C.B does not collect, store, or share personal data. It only processes necessary user inputs for moderation and club management.

2. Data Usage

All collected data is used solely for bot functionality, including moderation tasks and club-related automation.

3. Data Protection

No personal data is stored or shared with third parties. All interactions remain private within the server.

4. User Rights

Users can request data removal or restriction of bot interactions at any time by contacting the server administrator.

5. Security Measures

The bot follows strict security protocols to prevent unauthorized access or misuse of data.

6. Updates to This Policy

This policy may be updated as needed. Continued use of the bot implies agreement with the latest version.

For questions or concerns, contact the server admin.
